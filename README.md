# ChatProject
GUI와 Web을 이용한 채팅 프로젝트입니다.

# Description
+ 개발 기간: 2022.11.08 ~ 2022.11.16
+ 참여 인원: 5명
+ 사용 기술
  + Java, Javascript
  + Oralce 18c
  + Eclipse
+ 담당 구현 파트
  + GUI 기반 화면 구축
  + Oracle DB 연결 및 테이블 설계
  + 로그인 및 회원가입
  + 서버 채팅 내역 관리
